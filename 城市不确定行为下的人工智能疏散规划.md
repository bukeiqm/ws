# 不确定行为动态下的人工智能城市疏散规划研究

在不确定行为动态下，基于人工智能的城市疏散规划日益受到现代城市的依赖。这些系统在自然灾害、大规模事故和极端天气事件发生时，能够辅助制定紧急疏散计划。此类系统通常包含人群移动预测模块、动态路径规划算法以及应急资源分配策略。

然而，疏散规划面临的一大挑战是，人们在压力下的行为具有高度不确定性。人们可能做出非理性行为，如盲目跟随群体、突然改变路线或无视官方指导，这些不确定性会显著降低人工智能生成的疏散计划的有效性。

本研究旨在开发适用于不确定人群行为下的城市疏散数学和计算模型，并设计人工智能辅助策略，以提高疏散效率和安全性。

## 一、建立不确定人群行为动态模型

构建数学模型描述紧急情况下城市中疏散人员的移动情况，模型需包含以下核心要素：



1. 个体行为差异（例如，恐慌、犹豫、风险规避）

2. 群体效应（例如羊群效应或领导者 - 追随者模式）

3. 因拥堵或感知危险而进行的路线切换

4. 压力下决策质量的下降

可使用基于代理的建模、随机微分方程、马尔可夫过程、元胞自动机或其他合适的建模框架，需明确陈述所有假设并论证建模方法选择的理由。

### （一）个体速度和位移建模

速度 / 加速度由目标驱动项、群体交互、拥堵规避三部分组成，相关参数及公式如下：



* 速度 / 加速度核心公式：速度 / 加速度 = 目标驱动项 + 群体交互 + 拥堵规避

* 群体不确定性项：$\sigma(p_{i}(t)) d W t$


  * $p_{i}(t)$：心理状态函数（表示恐慌与压力）

  * $\sigma()$：激活函数，选择 $tan h (x)=\frac{e^{x}-e^{-x}}{e^{x}+e^{-x}}$（取值范围 (-1,1)，对称控制行为，如速度方向调整、左 / 右偏移、行动偏差）

  * $d W t ~ N(0, d t)$：服从正态分布的随机变量（均值为 0，方差为 dt，可通过代码离散生成）

* 人群密度与压力关系：$\rho^{around }(t)=\frac{N_{i}(t)}{A_{i}}$


  * $N_{i}(t)$：个体附近半径为 R 的人数

  * $A_{i}=\pi \cdot R^{2}$：感知范围面积

  * 规律：人密度越大，压力越大

* 危险系数：$D_{i}(t)$


  * 规律：危险系数越高，压力越大，越容易产生不确定行为导致错误判断

  * 影响因素：



| 影响因素   | 具体说明            |
| ------ | --------------- |
| 与危险源距离 | 越近越危险           |
| 环境刺激   | 目视烟雾、温度、声响带来的刺激 |

### （二）群体交互加速度 $a_{i}^{int }(t)$

表示人受到周围人的影响的加速度，包含拥挤碰撞效应和羊群效应（领导者 - 追随者模式），不同人群（邻居、陌生人、领导者、弱势群体）参数不同：



| 参数        | 说明                              |
| --------- | ------------------------------- |
| 当前距离      | 两个人现在的实际距离                      |
| 安全舒适距离    | 人感到安全、不会碰到对方的距离，可根据年龄、身高、拥挤情况调整 |
| A（力的强度参数） | 人对碰撞的敏感度；紧急情况可增大                |
| B（衰减参数）   | 排斥力随距离衰减的速度；小 B→快速衰减，远处几乎无力     |
| 方向单位向量    | 力的方向：从邻居指向自己，保证推动自己远离对方         |



* 特殊人群参数调整：


  * 领导者：C 值较大

  * 老人和小孩等弱势群体：推力小，A 值较小

  * B 参数参考：Helbing 等经典社会力模型文献，0.1\~0.5m 对普通成年人模拟合理；紧急疏散时 B 小一些→非常靠近时强烈避让，B 大一些→温柔推开

### （三）环境作用加速度 $a_{i}^{e n v}(t)$

$a_{i}^{e n v}(t)=\sum_{k=1}^{m} g_{i k}(t)$，其中 k 遍历环境中的火源、烟雾、坍塌、栏杆等，$g_{i k}(t)$ 为第 k 个环境对象对 i 个体的作用力（二维向量）：



| 参数                 | 说明                  |
| ------------------ | ------------------- |
| 个体认为安全的最小距离（或危险阈值） | 个体可接受的安全距离临界值       |
| 个体到环境物体的实际距离       | 个体与环境危险物 / 障碍物的真实距离 |
| 力的强度               | 与障碍物硬度、危险程度正相关      |
| 衰减尺度               | 数值越大，作用力衰减越平缓       |
| 单位向量               | 指向远离障碍 / 危险的方向      |

### （四）期望速度与反应时间



* $v_{i}^{desired }(t)$：期望速度向量（指向出口）

* $v_{i}(t)$：当前速度向量

* $\tau_{i}$：个体反应时间（心理延迟），不同人群差异显著（老人慢，青壮年较快）

* 调节机制：类似 PID 调节方法，个体通过加速或减速使当前速度逐渐接近期望速度

* 最大速度：个体在极限状态下的最大移动速度
